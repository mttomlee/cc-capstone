# Intrution detection using machine learning models
Capstone Project for my study in Centennial College. Course code: CBER710

## Input Data
The dataset being used in this project is obtained from Intrusion Detection Evaluation Dataset (CICIDS2017) given by the Canadian Institute for Cybersecurity. The whole dataset package includes information of normal and malicious network traffic which is generated from common attacks.

Around 5% of the whole dataset was extracted for training and testing purpose, and the number of
normal and malicious traffic data is nearly the same. Our dataset contains 54313 data entries and 80
columns, which 79 columns are different network features, and one column indicates the class.
